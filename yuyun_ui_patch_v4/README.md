## 安装方法

### 方法一：完整覆盖（推荐新安装）
1. 备份您的网站文件和数据库
2. 将补丁包内的所有文件上传到网站根目录，覆盖原有文件
3. 访问网站，刷新浏览器缓存即可生效

### 方法二：老版本升级（推荐）
1. **重要：先备份网站文件和数据库！**
2. 将补丁包内的文件按目录结构上传到对应位置
3. 覆盖原有文件
4. 访问网站，系统会自动完成数据库迁移
5. 刷新浏览器缓存（Ctrl+F5）

### 虚拟主机安装说明
- 本补丁包支持虚拟主机环境
- 支持 PHP 7.0+ 版本
- 支持 MySQL 和 SQLite 数据库
- 无需特殊服务器配置
- 直接上传覆盖即可使用

## 数据库升级

本补丁包包含自动数据库迁移功能：
- 自动创建设置表（如果不存在）
- 自动添加用户表缺失列
- 自动添加反馈表 reply 和 replied_at 列
- 无需手动执行 SQL
