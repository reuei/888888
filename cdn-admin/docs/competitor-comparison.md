# CloudShield CDN 竞品对比

## 市场定位

**比鹿云盾更开放，比 SCDN 更经济。**

CloudShield CDN 面向需要**自主可控、源码交付、私有部署**的中小企业、IDC 服务商、代理商以及有二次开发能力的团队。与鹿云盾、SCDN 等 SaaS 化平台相比，CloudShield CDN 提供完整后端源码，可部署在自有服务器或 EasyPanel 虚拟主机上，避免被平台绑定与高额流量费用。

## 为什么选择 CloudShield CDN？

### 1. 告别平台绑架

鹿云盾与 SCDN 均为闭源 SaaS，用户域名、证书、配置、数据均托管在平台。一旦平台调整价格、限制功能或停止服务，业务迁移成本极高。CloudShield CDN 源码交付，数据与控制权完全属于用户。

### 2. 更低的长期成本

SaaS 平台通常按流量计费的边际成本较高，对于流量型业务（视频、下载、游戏更新）会形成持续支出。CloudShield CDN 一次性授权后，用户仅需承担自有服务器与带宽成本，规模越大越划算。

### 3. 更适合代理商与 IDC

内置 S 端管理后台、B 端商户后台、代理分销、邀请码、套餐管理、资金流水、结算记录等模块，可直接作为面向下游客户的 CDN 服务平台，而无需从 SaaS 平台采购代理。

### 4. 完全可定制

前端采用 React 19 + TypeScript + Vite，后端采用 ThinkPHP 8.1，代码结构清晰，易于对接自有节点、支付网关、实名认证、工单系统等。

## 适用场景

- 中小型 CDN 服务商快速搭建自有平台
- IDC / 云主机商增值服务
- 企业内部站点加速与防护
- 开发者学习、二开、私有化项目
