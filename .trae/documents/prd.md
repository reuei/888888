# 语云科技企业官网 - 产品需求文档 (PRD)

## 1. 产品概述

语云科技（Yuyun Technology）中国企业官网及后台管理系统 - 一款面向全球的企业级云计算服务提供商官方网站，提供公司介绍、产品展示、客户服务、用户中心等完整功能，支持多模板切换和后台全配置管理。

**目标用户**: 企业客户、个人开发者、合作伙伴
**市场价值**: 展示语云科技品牌形象，提供产品服务入口，建立客户信任体系

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 浏览网站 | 查看所有公开页面内容 |
| 注册用户 | 邮箱注册/验证码登录 | 登录用户中心、提交工单、提交建议举报 |
| 管理员 | 后台分配 | 配置所有网站内容、管理用户、管理工单、管理模板 |

### 2.2 功能模块

#### 前台网站模块
1. **首页 (index.php)**: 轮播图、合作伙伴横滚、公司分布地图、业务产品展示、员工卡片、资质证书展示
2. **关于我们 (about.php)**: 公司信息、详细介绍、地图定位、联系方式
3. **公司简介 (intro.php)**: 公司发展历程、企业文化、团队介绍
4. **产品介绍 (products.php)**: 产品列表、产品详情、价格方案
5. **联系我们 (contact.php)**: 联系表单、客服入口、地址信息
6. **合作伙伴 (partners.php)**: 合作伙伴展示、合作申请
7. **国际版跳转**: 跳转至 https://cloud.loveym.cloud
8. **用户中心 (user/**): 注册登录、个人信息、工单系统、建议举报

#### 后台管理模块
1. **仪表盘**: 数据概览、快捷操作
2. **站点设置**: LOGO、名称、备案号、电话、SEO配置
3. **内容管理**: 轮播图、合作伙伴、友情链接、资质证书
4. **产品管理**: 产品CRUD、分类管理
5. **模板管理**: 模板选择、上传模板、模板配置
6. **用户管理**: 用户列表、权限管理
7. **工单管理**: 工单处理、回复工单
8. **员工管理**: 员工卡片CRUD
9. **系统设置**: 邮件配置、系统更新/补丁

## 3. 核心流程

### 3.1 用户访问流程
```
访客 → 首页 → 浏览页面 → (注册/登录) → 用户中心 → 提交工单/建议
                    ↓
              联系客服/查看产品 → (购买意向) → 联系销售
```

### 3.2 管理员流程
```
管理员登录 → 仪表盘 → 选择功能模块 → 编辑内容 → 保存发布
                                    ↓
                              模板管理 → 切换/上传模板 → 预览效果
                                    ↓
                              工单管理 → 查看工单 → 回复处理
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**: 深蓝 #0066CC + 科技蓝 #00A8E8
- **辅助色**: 橙色 #FF6B00 (强调色/电话号码)
- **背景色**: 白色 #FFFFFF / 浅灰 #F5F7FA / 深黑 #0D1117 (页脚)
- **文字色**: 深灰 #1F2937 / 中灰 #6B7280
- **按钮风格**: 圆角渐变按钮、hover阴影提升
- **字体**: 中文使用思源黑体/Noto Sans SC，英文使用Inter/Roboto
- **布局**: 现代卡片式布局、大量留白、响应式栅格
- **图标**: Font Awesome 6 / 自定义SVG图标库
- **弹窗**: 毛玻璃效果Modal、居中弹出、遮罩层、动画过渡
- **动画**: 渐入动画、滚动触发、悬停微交互

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero轮播 | 全宽轮播、渐变遮罩、文字叠加、CTA按钮 |
| 首页 | 合作伙伴 | 无限横滚动、灰色Logo、hover彩色、箭头指示 |
| 首页 | 产品服务 | 6列网格卡片、图标+标题+描述、底部CTA |
| 首页 | 全球地图 | SVG世界地图、脉冲标记点、Tooltip信息卡 |
| 首页 | 员工卡片 | 圆形头像、姓名职位、社交链接、横排展示 |
| 导航栏 | 顶部导航 | Logo左、菜单中、右侧固定联系组、sticky |
| 弹窗组件 | Modal | 毛玻璃背景、圆角容器、关闭按钮、入场动画 |
| 右侧悬浮 | 联系工具栏 | 垂直排列、图标+文字、hover展开、固定定位 |
| 页脚 | Footer | 三段式结构、Logo+电话、链接分组、备案信息 |

### 4.3 响应式设计
- **桌面端** (>1200px): 完整布局、多列网格、完整导航
- **平板端** (768px-1200px): 自适应列数、折叠次级导航
- **移动端** (<768px): 汉堡菜单、单列布局、触控优化

### 4.4 特色效果
- 3D CSS变换卡片效果
- 粒子/网格动态背景(可选)
- 滚动视差效果
- 数字计数器动画
- 打字机效果(Hero区域)
- 地图标记点脉冲动画
- 图片懒加载+淡入

## 5. 技术约束
- 运行环境: PHP 7.4+ / MySQL 5.7+(可选) / Apache/Nginx
- 兼容性: 现代浏览器(Chrome/Firefox/Safari/Edge最新两版)
- 部署方式: 虚拟主机直接上传
