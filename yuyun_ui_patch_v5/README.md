# 语云科技官网 UI 补丁包 v5

## 补丁包说明

本补丁包是语云科技官网的第五份 UI/功能升级补丁，适用于在虚拟主机上一键上传覆盖使用。无需二次修改，覆盖后即可运行。

## 更新内容

### 1. 修复后台管理保存后跳转白屏问题
- 重构 `redirect()` 函数，自动清理输出缓冲
- `headers already sent` 时自动降级为 HTML meta + JavaScript 重定向
- 在 `config.php` 中初始化输出缓冲，兼容虚拟主机环境

### 2. 修复后台站点配置无法保存并跳转白屏问题
- 新增 `setSettingSafe()` 安全保存函数，捕获数据库异常
- 保存失败时给出明确中文/多语言提示，不再直接白屏
- 兼容 MySQL / SQLite 双数据库

### 3. 修复切换语言时翻译文字不全问题
- 完善 8 种语言的后台管理界面翻译
- 新增首页拓展模块（统计数据、核心特性、客户评价、CTA 等）的翻译键
- 使用公开、通用、免费的在线翻译结果统一校对多语言文本

### 4. 修复地图显示问题
- 将百度地图替换为免费稳定的 OpenStreetMap 嵌入地图
- 移除对第三方 API Key 的依赖
- 虚拟主机无需额外配置即可正常显示

### 5. 修复首页地图显示
- 优化首页 SVG 世界地图布局与节点交互
- 添加节点悬浮 tooltip，显示城市与区域信息
- 修复移动端地图溢出与显示不完整问题

### 6. 修复首页弹窗公告不好看的问题（对标鲸发卡 pro）
- 重新设计欢迎弹窗：顶部渐变视觉区、流光动画、圆角卡片
- 关闭后自动收缩为右侧悬浮标签，点击可再次展开
- 支持多语言标题与内容，视觉风格更现代

### 7. 拓展并美化首页内容和模块及布局
- 新增数据统计区（客户数、SLA、节点数、7×24 支持）
- 新增核心特性展示模块（弹性云、安全、网络、服务）
- 新增客户评价模块
- 新增 CTA 行动号召区
- 整体布局更紧凑、现代，响应式适配桌面与移动端

### 8. 美化轮播图
- 3D 翻转切换效果
- 进度条动画（5 秒自动切换）
- 粒子背景与电影感标题排版

### 9. 美化并升级全部内容
- 统一卡片、按钮、表单、提示样式
- 深色模式视觉优化
- 增加滚动触发的数字增长动画
- 整体品牌色与渐变层次更协调

## 安装方法

### 方法一：完整覆盖（推荐）
1. **升级前请务必备份网站文件和数据库！**
2. 将补丁包内所有文件上传到网站根目录，覆盖原有文件
3. 访问网站，刷新浏览器缓存（Ctrl + F5）即可生效

### 方法二：使用升级脚本（推荐老版本升级）
1. **升级前请务必备份网站文件和数据库！**
2. 将补丁包文件上传到网站根目录并覆盖
3. 浏览器访问 `upgrade_v5.php`
4. 根据提示完成数据库表与默认设置检查
5. 升级完成后**删除** `upgrade_v5.php`
6. 刷新浏览器缓存

### 虚拟主机安装说明
- 本补丁包支持虚拟主机环境，上传即用
- 支持 PHP 7.0+ 版本
- 支持 MySQL 5.6+ 与 SQLite 3 数据库
- 支持 Apache / Nginx（Apache 建议保留 .htaccess）
- 无需特殊服务器配置
- 无需额外的 API Key 或在线服务授权

## 数据库升级

本补丁包包含自动数据库迁移逻辑（也可通过 `upgrade_v5.php` 手动执行）：
- 自动检测并创建/修复必需的表结构
- 自动初始化缺失的默认站点设置
- 不会覆盖或删除已有数据
